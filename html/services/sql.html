SQL Injection




Job Description

SQL injection is a web security vulnerability that allows an attacker to interfere with the queries that an application makes to its database. The impact SQL injection can have on a business is far-reaching. A successful attack may result in the unauthorized viewing of user lists, the deletion of entire tables and, in certain cases, the attacker gaining administrative rights to a database, all of which are highly detrimental to a business.
